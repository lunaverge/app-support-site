About DockerMon Pro

DockerMon Pro helps authorized users monitor and manage Docker containers on SSH-accessible servers they own or administer.

Common Setup Checks

  • Make sure the Docker server is reachable from your device.
  • Confirm the SSH host, port, username, and password are correct.
  • Confirm Docker is installed and available on the remote server.
  • Confirm your SSH user has permission to run Docker commands.
  • Allow notification permission only if you want monitoring alerts.
