DockerMon Pro Privacy Policy

Effective date: June 21, 2026

Overview

DockerMon Pro is designed to help users monitor and manage Docker containers on servers they own or are authorized to access. The app does not operate a developer-controlled backend service and does not sell user data.

Information Stored Locally

The app may store the following information locally on your device:

  • Server host, port, username, grouping, and display preferences.
  • SSH passwords or credentials in secure storage / Keychain.
  • Known SSH host key fingerprints.
  • Local audit logs for container and image operations.
  • Language, notification, and monitoring preferences.

Network Connections

DockerMon Pro connects only to servers configured by the user. Docker status, logs, images, and command output are retrieved from the user's server through SSH. This information is not uploaded to a developer-controlled server.

Third-Party Services

The app does not use third-party analytics SDKs, advertising SDKs, payment processors, AI services, or developer-operated authentication services.

Notifications

Notification permission is requested only when the user enables background monitoring or sends a test notification. Notifications are used to alert the user about selected container monitoring events.

Data Deletion

Users can remove stored server configurations and related local data from within the app. Users may also delete the app from the device to remove locally stored app data managed by iOS.
